🆘 Troubleshooting

❌ Issue: Settings not saving

  • Check browser console for errors (F12)
  • Make sure all required fields are filled
  • Try clearing browser cache (Ctrl+Shift+Del)
  • Close and reopen Settings panel

❌ Issue: Preview doesn't show changes

  • Click 🔄 Refresh Preview button
  • Close preview window and reopen it
  • Check if settings actually saved (see success message)
  • Hard refresh page: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

❌ Issue: Invoice creation fails

  • Check backend is running (look for "API" green indicator on Dashboard)
  • Fill all required fields in the form
  • Check browser console for error messages
  • Try reloading Dashboard

❌ Issue: Generated invoice doesn't use settings

  • Wait 2-3 seconds after saving settings
  • Refresh Dashboard page (F5)
  • Create invoice again
  • Check settings were actually saved
